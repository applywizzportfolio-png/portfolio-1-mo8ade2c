## 🚀 Key Features

- **3D Interactive UI**: Built with Framer Motion for immersive mouse-tracking tilt effects and page transitions.
- **Dynamic Background**: Custom-built DataFlowBackground simulating real-time data streaming patterns.
- **Micro-Animations**: 3D rotating icons for tech stack and certifications.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion
- **Icons**: Lucide React (3D-rendered)
- **State Management**: React Query (TanStack)
- **Theming**: Custom HSL-based Dark/Light mode system

## 📂 Project Structure

- `/src/components/3d`: High-performance reusable 3D components.
- `/src/components`: Domain-specific sections (Hero, About, Experience, etc.).
- `/src/pages`: Main application routes (Index, NotFound).
